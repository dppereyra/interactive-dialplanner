Compare

Activity

:

Compare

Description

:

Use this Activity to compare two values or variables.

Icon

:

 
Parameters

:

Label

:

 Name of label for the Activity in the call flow.  This is unique name in the call flow project.
Valuable 1, Valuable 2

:

Variables that will be compared.
- Type: variables, string

Operator

:

 To specify how to compare two values.  It may be:
= - equal
> - larger
< - smaller
!= - not equal

Output Node

:

If the condition of compare is true, the flow will go to next step via "Yes".  Otherwise, the flow will go to next step via "No".
