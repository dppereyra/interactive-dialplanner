Activity

:

Hangup

Description

:

Use this Activity to hangup the calling channel.  It is the end of the call flow.

Icon

:
Parameters

:

Label

:

 Name of label for the Activity in the call flow.  This is unique name in the call flow project.
Hangup Code

:

 If the value is given, the channel's hangup cause will be set to the given value.

From voip-info.org:
Very important!!! The <causecode> parameter is the ISDN causecode and not SIP. So, for example, in order to get Asterisk to hang up a call with the SIP cause 503 you'll need to call hangup(42).

- Type: variable, number
