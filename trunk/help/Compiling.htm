Compile Call Flow

You can compile the Interactive call flow to the Asterisk Dial Plan.

To compile the call flow:

  1.  Click Project > Make.  The application will ask you to save the file.  The call flow will be compiled after saved.



  2. Compiled information will show on the Output box.  The Dial Plan will be saved if compile success.



  3. If the Remote Update is enabled, the dial plan file will be uploaded automatically after compiling.
