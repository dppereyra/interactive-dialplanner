Read

Activity

:

Read

Description

:

Use this Activity to read variable in the form for DTMF tones as pressed by the caller.

Icon

:
Parameters

:

Label

:

 Name of label for the Activity in the call flow.  This is unique name in the call flow project.
Result

:

 Variable where the user's response will be stored.
- Type: variable, string
Entry Message

:

 File to play before reading digits.
- Type: variable, string
Options

:

  May be 'skip' to return immediately if the line is not up, or 'noanswer' to read digits even if the line is not up.
- Type: string
Max Digits

:

 Maximum acceptable number of digits.
Maximum silence

:

 Number of attempts will be made in the event no data is entered.
Number of retries on error

:

 Timeout in seconds. If greater than 0, that value will override the default timeout.

Output Node

:

 Done
