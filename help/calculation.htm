Label

 Name of label for the Activity in the call flow.  This is unique name in the call flow project.
Calcualte

:

Perform calculation on Number1 to Number2. Valid ops are:
+,-,/,*,%,<,>,>=,<=,==
and behave as their C equivalents.
- f, float
- i, integer
- h, hex
- c, char

The Number1 and Number2 can be input number value directly, but also using variable as a input value.  For how to use variable in the call flow diagram, see Using Variable

- Type: variable, number

Result

:

 Variable for storing the calculation result.  For how to use variable in the call flow diagram, see Using Variable
- Type: variable
