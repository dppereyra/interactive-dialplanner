Play Message

Activity

:

PlayMsg

Description

:

Use this Activity to playback a sound file.  User can choose whether to allow the caller stop playing the message by pressing key.

Icon

:

   
Parameters

:

Label

:

 Name of label for the Activity in the call flow.  This is unique name in the call flow project.
Message

:

 File name of the message.
- Type: variable, string
DTMF

:

 If the DTMF check box selected, the Activity will stop playing if user pressing key. Otherwise, the message will play whole sound file and when complete.

Output Node

:

Done

:

 If the message plays to completion without any interrupting digit, the call exits out of the Done node.
DTMF

:

 If the message plays to completion with any interrupting digit, the call exits out of the DTMF node.
