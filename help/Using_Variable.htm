Using Variable

Interactive Dialplanner allow Interactive Engineer to define variables for the call flow to store the value during the Asterisk dialplan execution.  There are two types of variables:

  1. User defined variable - variable shall be defined by user.
  2. System variable - system-related variables that pre-defined by the Asterisk PABX.

Define variable

  1. Click Project > Variables.  A dialog box appears.

  2. In the Variable dialog box, click "Add" to add variable.  Another dialog box appears.

  3. Specify the variable name.  Then click "OK" button.

  4. The variable added.  Click "OK" button to exit.  If another variable you want to define, continue click "Add" button.

Assign variable into Activity - Using Record Activity as Example

  1. Double click the Activity.  The dialog box appears.

  2. Click "Variables" button to open the dialog box.

  3. Now we want to assign "variable1" into the "File Name" of Record Activity.  You can hold the "variable1" and drag the variable into "File Name" Field.

  4. The "variable1" had been assigned to the specific location.
