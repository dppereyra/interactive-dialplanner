SubProc

Activity

:

SubProc

Description

:

Use this Activity to call a current extension sub-flow and get the return from the flow.

Icon

:

  
Parameters

:

Label

:

 Name of label for the Activity in the call flow.  This is unique name in the call flow project.
Select Sub-Flow

:

All sub-flow will be listed on the list box you can be selected.  The selection will be the flow that to be directed to.

Arguments

:

Arguments pass into the sub-procedure.  E.g.  ${ARG1}, ${ARG2}, etc

Result

:


The sub-procedure defines one or more exit nodes, each by a SubProcEnd Activities. Corresponding exit nodes are created manually in the SubProc when the Sub-Procedure is set on the property page. When the sub-procedure executes one of the SubProcEnd controls the SubProc exits out the corresponding exit node.

Output Node

:

Done
