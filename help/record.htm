Record

Activity

:

Record

Description

:

Use this Activity to record from the current channel to a file.

Icon

:
Parameters

:

Label

:

 Name of label for the Activity in the call flow.  This is unique name in the call flow project.
Filename

:

 Record file name
- Type: variable, string
Format

:

 Format of the recording.
Silent

:

 Seconds of silence allowed before the recording is stopped.
Max Duration

:

 Maximum recording duration in seconds.
Options

:

 May be 'skip' to return immediately if the line is not up, or 'noanswer' to record even if the line is not up.

Output Node

:

 Done
