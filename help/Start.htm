Start

Description

:

Each call flow contains an Activity of this type.  It is the entry point of the call flow.  The action it points to is the first Activity that will be processed when the call flow is called.  This Activity is used to identify each section of the flow in the call folw diagram by extension pattern.  Therefore it can simplify the implementation of the Interactive CallFlow Builder.
