Authenticate

Activity

:

Authenticate

Description

:

Use this Activity to authenticate a user with a dialed passcode.  It requires a user to enter a password in order to continue execution.

Icon

:

Parameters

:

Label

:

 Name of label for the Activity in the call flow.  This is unique name in the call flow project.
Password

:

 The password thet will be matched with user input in order to continue execution.
- Type: variable, string

Output Node

:

Done

:

 The call flow will exit via Done node if the authenticate successful.
Fail

:

 The call flow will exit via Fail node if the authenticate failed.
